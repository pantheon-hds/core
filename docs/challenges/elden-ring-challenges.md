# Elden Ring — Community Challenges

## Gold (baseline — all Steam achievements required first)

Automatically assigned via Steam API when 100% achievements unlocked.

---

## Platinum tier

### 1. Naked & Afraid
Complete the game wearing no armor. No chest, no helm, no legs, no gauntlets.
Weapons and shields allowed.
*Verifiable via save file — armor slots empty at all boss kills.*

### 2. Flask Free
Defeat any 5 Great Enemies without using a single flask during the entire run.
Crimson, Cerulean, and Wondrous Physick all count.
*Verifiable via video — flask count visible in HUD.*

### 3. One And Done
Complete the game with only one weapon from start to finish.
No switching. Same weapon must land the killing blow on final boss.
*Verifiable via video — weapon slot visible throughout.*

### 4. Speed of the Erdtree
Complete the game in under 3 hours from character creation to final boss death.
*Verifiable via video — timer visible, no cuts.*

---

## Diamond tier

### 5. The Purist
Complete the game with no Ashes of War equipped on any weapon at any time.
Default movesets only.
*Verifiable via save file + video.*

### 6. Level One Wretch
Defeat Malenia, Blade of Miquella at Soul Level 1.
No summons. No co-op.
*Verifiable via video — level visible in stats menu.*

### 7. Ghost Run
Complete the game without killing any optional enemies.
Only bosses required for progression may be killed.
*Verifiable via video — high scrutiny required.*

### 8. Rune Ascetic
Complete the entire game without spending a single rune on leveling up.
Runes may be used for purchasing items only.
*Verifiable via save file — level remains at starting value.*

---

## Legend tier (community vote required)

### 9. True No Hit
Complete the game from start to final boss without taking a single hit.
All bosses required for main ending must be defeated.
No summons. No co-op. Full uncut stream required.
*One of the hardest challenges in the game. Very few have done it.*

### 10. Blindfolded Margit
Defeat Margit, the Fell Omen — the first major boss — completely blindfolded.
Verified by community. Streamed live with camera showing the blindfold.
*Not about completing the game. About pure muscle memory and mastery.*

---

## Notes
- All challenges require uncut video proof
- HUD must be visible at all times
- End-game stats screen must be shown
- Diamond and above go through blind anonymous voting (5 judges)
- Legend tier requires community open vote
