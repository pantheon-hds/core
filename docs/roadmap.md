# Roadmap

## Philosophy

One step at a time. One game. One community. Done right before moving forward.

---

## Phase 1 — Foundation
*Target: first 1,000 players*

**Platform**
- [ ] Web application launch
- [ ] Steam OpenID login
- [ ] Gold rank auto-assignment via Steam API
- [ ] Player profile with statue display
- [ ] Basic challenge list for Elden Ring
- [ ] Video submission system
- [ ] Manual review by founder

**Community**
- [ ] Launch with Elden Ring only
- [ ] Post on r/Eldenring — honest, no hype
- [ ] Ask community for 20-30 challenge ideas
- [ ] Find 3-5 small streamers to try the platform
- [ ] First Legend candidate

**Design**
- [ ] Welcome screen — Honor · Democracy · Skill
- [ ] Dashboard with rank and active challenges
- [ ] Pantheon page
- [ ] Hall of Statues on player profile

---

## Phase 2 — Community engine
*Target: first 10,000 players*

**Platform**
- [ ] Blind anonymous voting system (5 random judges)
- [ ] Appeal system
- [ ] Peer review replaces founder review
- [ ] Community challenge creation and voting
- [ ] Supporter Pack — cosmetic only, zero gameplay effect
- [ ] Open source code published on GitHub

**Community**
- [ ] Add second game — Hollow Knight
- [ ] First platform festival with prizes
- [ ] First official Legend statue granted
- [ ] Partner with Razer or similar for festival prizes

**Mobile**
- [ ] Simplified mobile web version
- [ ] View rank, vote, follow challenges on the go

---

## Phase 3 — Global scale
*Target: 1,000,000 active players*

**Platform**
- [ ] Save file parsing for automated Platinum verification
- [ ] Steam Stats API integration for challenge conditions
- [ ] Regional communities — China, Japan, USA, Europe
- [ ] Multi-language support
- [ ] Festival calendar — seasonal events

**Community**
- [ ] 10+ games supported
- [ ] Developer partnerships — studios pay to be featured
- [ ] Community-gifted prizes between players
- [ ] Streamer widget for OBS — live rank display

**Desktop**
- [ ] Standalone Hall of Fame desktop app
- [ ] 3D statue viewer
- [ ] Immersive pantheon experience

---

## Phase 4 — Legacy
*No deadline — this is permanent*

**Platform**
- [ ] Full decentralization — community owns the platform
- [ ] Developer API — any game can integrate natively
- [ ] Global festival — annual event, all games, all regions
- [ ] Pantheon Archive — permanent history of every Legend ever granted

**Philosophy**
- Honor · Democracy · Skill — unchanged from day one
- No advertisements. Ever.
- No pay-to-win. Ever.
- Community decides everything that matters.

---

## What we will never do

- Sell ranks or skip verification for money
- Accept advertising that influences platform content
- Sell the platform to a corporation without community consent
- Remove a Legend statue without a community vote
- Change the three core principles
